This section decides the type of items. This is depended on MMOItems config. 

If the category is not configed in Mmoitems, EnchantSlots WILL NOT recognise it and the lore section will not be displayed. 

```
categories:
- SWORD
- TOOL
- DAGGER
- SPEAR
- HAMMER
- BOW
- CROSSBOW
- ARMOR
- GREATSWORD
- LONGSWORD
- KATANA
- AXE
- GREATAXE
- HALBERD
- GREATHAMMER
```
